README
=================================================================================

.. contents:: **Daftar Isi**

nnn adalah terminal file manager. 

Install di Ubuntu
---------------------------------------------------------------------------------

::

    $ sudo apt install nnn

Manual
---------------------------------------------------------------------------------

Navigation

    - q: quit

Files

    - n: buat file atau folder
    - d: menampilan info detail sebuah file/folder
    - e: open file
